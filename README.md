# Apex Auto

**A responsive dealership showcase website — dark/light theming, inventory filtering, financing, and custom animations.**

🔗 **Live site:** https://apex-auto-83z.pages.dev

A multi-page website for a used-car dealership, built and deployed end to end as a self-directed project. Designed to be fast, mobile-friendly, and easy for a small dealer to actually use. It is tailored for a specific dealership that a good friend of mine works at.

## Features

- 🌗 **Dark / light theme toggle**
- 🔎 **Inventory filtering** to browse vehicles by type
- 📝 **Financing request form**
- ✨ Custom animations and transitions
- 📱 Fully responsive layout

## Tech stack

- **HTML / CSS / JavaScript**
- Deployed via **Cloudflare Pages**, connected to this GitHub repo for automatic builds on push

## Deployment

Pushing to `main` triggers an automatic Cloudflare Pages build and deploy — no manual steps.

## Status

Self-directed portfolio build. Designed as a template for a real dealership use case.
